# LaboratoryGame
 
## Description

Small plataformer made by two videogame development students.

## Controls

WASD for movement

SPACE for jump

F1 show collisions

F10 God mode (unkillable and infinite jump)

L to load save file

F5 Reset save file

F4 for recharge scene
